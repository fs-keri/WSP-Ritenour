Component Library

FOOTER

WHAT I HOPE TO PRACTICE/LEARN?

This component was selected so that I can practice column layouts in the footer area that are more complicated that the simple one line copyright information. I struggle with elements that need to be spread out into column type.

HOW THIS RELATES TO WHAT I HAVE/WILL LEARN IN WDD

The footer is not the end of a site and can contain information that will allow the users to delve deeper into the site. It provides easy navigation on longer pages that may require the user to scroll. It can increase search engine rankings.

HOW I MIGHT BUILD THIS?
  1. Create the component as a <footer>
  2. Create a <navigation>
  3. Add an <h5>
  4. Create a <ol> and add <li> for each item below the heading
  5. Repeat steps 2 thru 4 for the other two lists
  6. Create <address> for the contact information
  7. Add an <ol> and <li> for the social media
  8. Add another <footer> and <p> for the copyright
